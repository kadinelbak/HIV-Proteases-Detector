Numeralizing Chemistry

Morgan Fingerprints: From Atoms to Bit-Vectors
Sample Fingerprint
Each atom's circular neighborhood (radius 0–2) hashes to multiple scattered bit positions in the 2048-bit fingerprint
CH₃–C(=O)NHC₆H₃OH
2048-bit fingerprint · folded to 32×32
CH₃ — methyl
C=O — carbonyl
NH — amine
C₆H₃ — ring
OH — hydroxyl

Morgan Fingerprints encode circular neighborhoods of atoms into bit-vectors that the MLP can actually "read." This bridges the gap between raw chemistry and machine learning.

  • SMILES: Symbolic encoding of molecular structure (atoms, bonds, rings)
  • Morgan FP: 1024-bit hash of atom neighborhoods (radius=2, diameter=4 bonds)
  • Why 1024 bits? Sufficient hashing space for common organic chemistry
  • Circular neighborhoods: Each bit captures local chemical context
"Each atom or bond in a SMILES string lights up a unique pattern in the bit grid."
$$b_i = \text{hash}(\text{env}_r(a_i))\bmod N$$
Bit $ b_i $ encodes atom $ a_i $’s circular neighborhood at radius $ r $

The “Needle in the Haystack” Problem

Standard accuracy is a lie here.
Class Distribution
Click to see why accuracy is misleading!

The dataset is highly imbalanced: 3.5% Active, 96.5% Inactive. A model that always predicts "Inactive" gets 96.5% accuracy—but finds zero drugs. We need specialized loss functions.

"If you always guess 'Inactive', you win the accuracy game but lose the drug discovery race."
$$\text{Acc}_\text{random} = \max(p,\,1-p) \approx 96.5\%$$
Trivial accuracy from always predicting the majority class

Learning the Chemical Grammar

A Deep MLP with ReLU & Dropout
Training/Validation Loss
MLP Schematic & Backprop Visualizer
Step: forward pass; Backprop: error signal

A multilayer perceptron (MLP) learns to map molecular fingerprints to inhibition probability. Each layer transforms the representation, extracting higher-order features. Backpropagation enables the network to learn from errors and improve predictions.

  • Input layer: 1024 bits (Morgan fingerprint)
  • Hidden layers: 512 → 256 units with ReLU activation
  • ReLU: $ \text{ReLU}(x) = \max(0, x) $ — enables non-linear learning
  • Dropout(0.5): Randomly zeros 50% of activations during training to prevent overfitting
  • Output: Single logit passed to sigmoid → probability [0, 1]
"The MLP learns the hidden grammar of chemistry—one weight update at a time."
$$\mathbf{a}^{(l)} = \text{Dropout}(\sigma(\mathbf{W}^{(l)}\mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}))$$

Dropout

Dropout randomly disables ("drops") a subset of neurons during training, forcing the network to learn redundant paths. This prevents overfitting and makes the model more robust.

  • Randomly zeroes 50% of activations during training
  • Forces network to not rely on any single path
  • At inference, all nodes are used (no dropout)

ReLU

The ReLU (Rectified Linear Unit) activation function outputs zero for negative values and passes positive values unchanged. This enables non-linear learning and helps networks train faster.

  • Formula: $\text{ReLU}(x) = \max(0, x)$
  • Collapses negative weighted sums to zero
  • Keeps positive signals intact
$$\tilde{h} = h \odot m,\quad m_i \sim \text{Bernoulli}(1-p)$$
Dropout mask $ m $ zeros each neuron independently with probability $ p $

Balancing the Scales

Weighted Binary Cross-Entropy Loss
Confusion Matrix
Weighted Scale Demo
Inactive: 40,000 Active: 1,443 Penalty: 27x
Move the slider to balance the scale

We force the model to care ~27x more about missing an Active compound. The penalty slider shows how the scale balances as we increase the weight for Actives.

  • Binary Cross-Entropy (BCE): Standard loss for binary classification
  • Weighted BCE: Applies weight $ w $ to positive class losses to handle imbalance
  • pos_weight = 27: Rare actives matter 27× more than common inactives
  • Effect: Model penalized heavily for false negatives (missed drugs)
  • Trade-off: Higher false positive rate acceptable if it catches more true actives
"This engineering hack makes rare actives visible to the model."
$$\mathcal{L} = -[w \cdot y \log(\sigma(z)) + (1-y) \log(1-\sigma(z))]$$
where $ z $ is logit, $ \sigma $ is sigmoid, $ w \approx 27 $

The Future: Next Steps

Graph Neural Networks & Active Learning

This project demonstrates the transformative potential of deep learning for molecular property prediction. Next steps: Graph Neural Networks for richer 3D molecular representations, and active learning for iterative wet-lab feedback loops.

  • ▶ GNN: atoms as nodes, bonds as edges
  • ▶ Active learning: model picks next compound to test
  • ▶ Interpretability: which substructures matter?
